# 🌍 Travel
✈️ 旅行攻略 & 游记合集  


## 🍪 食用说明

- 📂 **guides/** : 存放攻略，偏实用，交通、美食、住宿推荐  
- 📂 **diary/** : 存放游记，偏日常记录，感受与故事  
- 📂 **photos/** : 部分旅行照片


## 💡 使用方式

- Markdown 格式编写，方便阅读  
- GitHub 在线直接看，或者 `git clone` 下来自用  
- 若有更新，欢迎 PR / Issue 分享更多旅行灵感⭐  


## ✨ 小目标

- 打卡更多国家和城市 🌏  
- 每次旅途都写点文字 📒  
- 让攻略和游记成为回忆的「地图」 🗺️
